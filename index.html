¿Qué es HTML?

HTML (HyperText Markup Language) es el lenguaje de marcado estándar para crear páginas web.

¿Para qué se utiliza HTML?

HTML se utiliza para estructurar el contenido en la web, permitiendo la inclusión de texto, imágenes, enlaces y otros elementos multimedia.
